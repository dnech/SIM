Chat

- Схема шифрования "end-to-end"
- Алгоритм шифрования AES256
- Групповое общение в комнатах
- Единный пароль для группы участников

Amnesia

- Схема шифрования "end-to-end"
- Алгоритм шифрования RSA (512, 1024, 2048, 4096), AES256
- Быстрое, максимально защищенное, одноразовое, общение с одним собеседником
- Нет истории общения

BlackOut

- Схема шифрования "end-to-end"
- Алгоритм шифрования RSA (512, 1024, 2048, 4096), AES256

- Адресная книга
- Сохраняется история сообщений
